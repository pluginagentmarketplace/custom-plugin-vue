# Skill Assessment & Evaluation

Evaluate your current technical skills, identify knowledge gaps, and get targeted recommendations for improvement. This command provides comprehensive skill assessment across all development domains.

## Usage

Use this command to:
- **Self-Assess**: Evaluate your proficiency in specific skills
- **Identify Gaps**: Discover what you need to learn
- **Track Progress**: Monitor improvement over time
- **Get Recommendations**: Receive targeted learning suggestions
- **Benchmark Against Roles**: Compare your skills to role requirements
- **Create Action Plans**: Plan skill development priorities

## Assessment Framework

### Proficiency Levels

**Level 1: Awareness**
- Basic understanding of concepts
- Can follow tutorials and examples
- Limited independent problem-solving
- Time to reach: 1-2 weeks

**Level 2: Beginner**
- Can complete simple tasks
- Understand core concepts
- Need guidance for complex problems
- Time to reach: 4-8 weeks

**Level 3: Intermediate**
- Can build complete projects independently
- Understand best practices
- Can solve most common problems
- Time to reach: 3-6 months

**Level 4: Advanced**
- Can architect solutions
- Handle complex challenges
- Can mentor others
- Can contribute to open source
- Time to reach: 1-2 years

**Level 5: Expert**
- Deep mastery of domain
- Can innovate and create new solutions
- Influential in community
- Can consult and teach
- Time to reach: 3+ years

## Self-Assessment Questions

For each skill, answer honestly:

1. **Can I explain this concept to others?**
2. **Can I implement this without looking at examples?**
3. **Can I identify and fix problems in existing code?**
4. **Have I used this in a real project?**
5. **Can I teach someone else this skill?**

## Quick Assessment

Ask Claude to help evaluate:
- "Assess my React skills - I've built 3 projects"
- "What's my level in Python? I've completed courses but no real projects"
- "Evaluate my DevOps knowledge - I use Docker but not Kubernetes"
- "Assess backend skills - experienced with Node.js and PostgreSQL"
- "What's my proficiency in system design?"

## Skills Gap Analysis

Once assessed, receive:
- **Current Skills**: Your strengths and expertise areas
- **Gap Analysis**: Skills below your target level
- **Priority Recommendations**: What to learn first
- **Resource Suggestions**: Courses, books, projects
- **Timeline Estimates**: How long to improve

## Skill Development Plans

Based on assessment, get:
1. **Short-term Goals** (1-4 weeks)
   - Quick wins to build confidence
   - Foundation skills to solidify
   - Immediate gaps to address

2. **Medium-term Goals** (1-3 months)
   - Core competencies to develop
   - Real project implementation
   - Depth in specialization areas

3. **Long-term Goals** (3-12 months)
   - Advanced proficiencies
   - Leadership capabilities
   - Expert-level mastery

## Benchmarking

Compare your skills against:
- **Role Requirements**: What's needed for your target position
- **Industry Standards**: Expectations for your experience level
- **Your Goals**: Personal career objectives
- **Peers**: General proficiency expectations

## Tracking Progress

Re-assess periodically to:
- Verify improvement
- Celebrate achievements
- Adjust priorities
- Update learning paths
- Stay motivated

## Integration with Other Commands

- Use **/learning-path** to create targeted improvement plan
- Use **/explore-roadmap** to understand role requirements
- Use **/career-guidance** to align with long-term goals
